How To

To upload a series of maps into Maptcha, you need to upload a CSV that contains a row for each map. There are 4 required fields, but more is encouraged.

Here's an example CSV file to get you started.

Map Metadata Fields Red means required

Title
A name for each map.
Entrance to San Francisco Bay California.
Date
When the map was made. A year is enough information.
1859
Dropbox_Image_URL
This must be a fully qualified URL (with the http://) to a public DropBox repository where Maptcha can access an actual image file. That means the URL needs to point at a .jpg or .tif. Please link to the largest possible image size.
http://dropbox.com/8467/8075467309_d1eb3cee97_o.jpg
Place_Names
Please note any cities or town names you can see on the map. Please separate each place name with a comma. The more the better!
San Francisco, Oakland, Sacramento
Scale
1: ??
50,000
Latitude/Longitude
This is always useful. Even if it's just the lat/lon of an approximate center point on the map, that's very helpful. To 4 decimal points is perfecto!
37.7706/-122.3782
Description
This may well come straight out of the catalog.
Entrance to San Francisco Bay California. From a Trigonometrical Survey under the direction of A.D. Bache Superintendent of the Survey Of The Coast Of The United States. Triangulation by R.D. Cutts Asst. & A.F. Rodgers Sub-Assts. Hydrography by the Party under the command of Lieut. Comdg. James Alden U.S.N. Asst. 1859. U.S. Coast Survey Office. Verified W.R. Palmer ... No. 36. Redd. Drng. by W.M.C. Fairfax, J. Lambert, & J.J. Ricketts. Engd. by J. Knight, A. Blondeau & G.B. Metzeroth. Electrotype Copy No. 2 by G. Mathiot U.S.C.S. (inset map) Sub-Sketch Of Entrance To San Francisco Bay. 1859.).
Notes
You might like to describe the literal condition of the map, etc.
Light paper chart-full margins, shows plate marks. Map is without color.
Keywords/Tags
There are no restrictions on which keywords or tags to use. The more the better! Please separate each keyword or tag with a comma. You can join two words together in a tag using quotes: "two words", one. You may also like to experiment with machine tags.
chart map, publisher: U.S. Coast Survey Washington
License
Make sure to let people know if/when/how they can share or reuse the map.
© Copyright 1998 Cartography Associates
Repository or Archive
Attribution to the correct source is very important to Maptcha. Please use the official name for the collecting institution.
David Rumsey Historical Map Collection
Map Maker
This might be an individual, or an organization. Consider using machine tags above to include more detail.
United States Coast Survey
Link
If this map has a home on the web, please link to it. Watch out for session-based URLs. They don't last very long.
http://www.davidrumsey.com/luna/servlet/detail/RUMSEY~8~1~207~20027:Entrance-to-San-Francisco-Bay-Calif?qvq=mgid:23358&mi=0&trs=15
Filename
Even though you've told us this in the URL above, this is a very useful reconciliation field for us.
8075467309_d1eb3cee97_o.jpg
Identifier(s)
If this map has an identifier in your database, or other sites on the web, please add it here. URLs also work.
id:davidrumsey=20720027
Map Width
Please include units. Please avoid abbreviations.
120 centimetres
Map Height
