How To

There are 4 basic steps to uploading a set of maps to Maptcha:

  1. Select a set of historical maps you'd like to upload to Maptcha, and place the digital versions somewhere public on the web. Public here just means somewhere online that isn't password-protected.
     
  2. Create a spreadsheet that describes each of the maps you've selected, a row in the spreadsheet for each map. You'll need to export that information as a CSV file from your spreadsheet, and upload it somewhere public on the web. There's a service called Dropbox that is handy for this sort of thing. See below for more information on what you need to include in the CSV.
     
  3. Tell Maptcha where that CSV file is located using the form above, and hit Next.
     
  4. Provide some simple geographical hints about the atlas, like, any particular map features (street-level, coastlines etc) and for city-name clues to help determine a bounding box that contains all the maps in the atlas. This will come in handy later, for volunteers who are placing the maps to begin in the right general area.

Creating A CSV Metadata File

Here's an example CSV file you can use as a template. You can open this example up in any spreadsheet program and modify it. You're most welcome to recycle this example file. All you have to make sure to do is keep the column names the same. None of the fields are controlled, so you can use whatever metadata makes sens to you, and you can make the filename you like, since Maptcha doesn't need that to be anything specific.

There are 3 required fields that must be specified for each map in your CSV:

Date
When the map was made. A year is enough information.
1859
Image_URL
This must be a fully qualified URL (with the http://) to a public file on the web where Maptcha can access an actual image file. That means the URL needs to point at a .jpg, .png or .tif. Please link to the largest possible image size.
http://dropbox.com/8467/8075467309_d1eb3cee97_o.jpg
Title
A name for each map.
Entrance to San Francisco Bay California.

There are several other recommended fields:

Place_Names
Please note any cities or town names you can see on the map. Please separate each place name with a comma. You can make this as general or specific as you like. The more the better!
San Francisco, Oakland, Sacramento
Scale
1: ??
50,000
Latitude/Longitude
This is always useful. Even if it's just the lat/lon of an approximate center point on the map, that's very helpful. To 4 decimal points is perfecto! Try using <a href="http://www.getlatlon.com/">getlatlon.com</a> if you need help generating your lat/lon.
37.7706/-122.3782

Notes
You might like to describe the literal condition of the physical map, etc.

Keywords/Tags
There are no restrictions on which keywords or tags to use. The more the better! Please separate each keyword or tag with a comma. You can join two words together in a tag using quotes: "two words", one. You may also like to experiment with machine tags.
chart map, publisher: U.S. Coast Survey Washington
