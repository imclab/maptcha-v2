How To

There are 3 basic steps to uploading a set of maps to Maptcha:

  1. Upload a bunch of huge maps to Dropbox and make a note of their URLs. We call this set of maps an "atlas".
  2. Upload a CSV to Dropbox that describes all the maps. There's more detail on the CSV here.
  3. Fill out this form and, hey presto!

Here's an example CSV file you can use as a template. (Please keep the column names intact.)

Map Metadata Fields

There are 3 required fields:

Title
A name for each map.
Entrance to San Francisco Bay California.
Date
When the map was made. A year is enough information.
1859
Image_URL
This must be a fully qualified URL (with the http://) to a public DropBox repository where Maptcha can access an actual image file. That means the URL needs to point at a .jpg or .tif. Please link to the largest possible image size.
http://dropbox.com/8467/8075467309_d1eb3cee97_o.jpg

And several other recommended fields:

Place_Names
Please note any cities or town names you can see on the map. Please separate each place name with a comma. The more the better!

Keywords/Tags
There are no restrictions on which keywords or tags to use. The more the better! Please separate each keyword or tag with a comma. You can join two words together in a tag using quotes: "two words", one. You may also like to experiment with machine tags.
